# Golf Tracker — AI-Powered Golf Coach

A Progressive Web App for tracking golf rounds, analyzing performance trends, and getting AI coaching tips.

## Features

- **Shot Logger** — Log every shot: club, distance, shot type, outcome. Review rounds with score vs. par.
- **Performance Dashboard** — Score trends, average distance by club, shot type breakdown, outcome distribution.
- **AI Golf Coach** — Chat-based analysis of your round data. Works offline with built-in analysis, or connect a Claude API key for live coaching.
- **PWA** — Install on your phone from the browser. Works offline on the course.

## Tech Stack

- React + Vite
- Recharts (data visualization)
- Lucide React (icons)
- vite-plugin-pwa (offline support)
- Claude API (optional, for live AI coaching)

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
# Serve the dist/ folder on any static host (GitHub Pages, Vercel, Netlify)
```
